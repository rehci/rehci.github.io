## How to Use

1. **Browse**: Use the navigation menu to explore different categories
2. **Search**: Use the search bar to find specific articles
3. **Read**: Click on any article to read the full content
4. **Explore**: Use tags and categories to discover related content

## Contributing

To add new articles, simply create a markdown file in the `content` directory with the following frontmatter:

\`\`\`yaml
---
title: Your Article Title
description: A brief description
category: Category Name
tags: [tag1, tag2]
date: YYYY-MM-DD
author: Author Name
image: /path/to/image.jpg
---
\`\`\`

Then write your content in Markdown format below the frontmatter.
